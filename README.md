스프링 부트 + JSP 서점&도서관 통합 사이트

## 📌 주요 기능 - LM도서
### 메인페이지
- 이벤트 이미지 슬라이드
- 추천 도서
### 로그인
- DB값 검증
- 아이디/비밀번호 찾기
- 로그인 유지
- 로그인 시 쿠키(Cookie) 및 세션(Session) 생성
### 회원가입
- 비밀번호 SHA-256 해시 함수 암호화
- 주소지 입력(선택)
### 마이페이지
- 회원정보 수정
- 찜한 도서
- 주문 내역(주문취소)
- 문의 내역
- 이벤트 참여 내역
### 도서구매
- 바로구매/장바구니
- 도서 선택(전체 도서목록/베스트셀러/신규도서/도서상세페이지)
- 수량 선택
- 배송지/포인트 입력
- 영수증 페이지
- 결제
### 이벤트
- 출석체크
- 댓글 이벤트
- 퀴즈 이벤트
